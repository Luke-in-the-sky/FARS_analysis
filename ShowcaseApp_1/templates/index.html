The data

I tested the Fatality Analysis Reporting System (F.A.R.S.) database to see if there are some suggestions we can make to help people stay safe. The following tree shows an example of possible outcomes, here is how it works: at each node a question is asked: if your answer is yes, follow the upper branch, otherwise follow the lower one.

The technicals

The tree was obtained by applying a Decision Tree Classifier to the F.A.R.S dataset. This is a great method for quickly finding out which features are the most important to look at. It is also a great way to handle categorical data. We can also ask by how much a set of features helps more than another, i.e. by how much one set gives a better predictability than another.
